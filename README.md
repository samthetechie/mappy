
Howto
=====
*Visit https://github.com/samthetechie/mappy

  $ git clone git://github.com/samthetechie/mappy.git
  $ cd mappy
  $ sudo virtualenv ./env
  $ source env/bin/activate
  $ pip install -r requirements.txt
